# Marlin — Roadmap

## V1 — Current

- [x] Single SKILL.md with four modes (swift, sharp, strike, sonar)
- [x] V3 DSL internal/external format + bridge
- [x] Three domain schemas: web-api, data-pipeline, agent-task
- [x] npx skills one-liner install
- [x] Works with Claude Code, Cursor, Antigravity, 40+ agents
- [ ] Benchmark results (in progress)

## V2 — Next

- [ ] npm package: `npm install marlin-skills`
- [ ] REST API: `POST /compress` → returns compressed prompt
- [ ] Deterministic compression — no LLM needed, runs offline
- [ ] One-liner install for Gemini CLI, Codex, OpenCode
- [ ] Domain schema expansion (more domains)

## V3 — Future

- [ ] Multi-agent pipeline support
- [ ] Agent-to-agent DSL passing protocol
- [ ] Schema versioning and migration
- [ ] SkillHQ and skills.sh registry listing
